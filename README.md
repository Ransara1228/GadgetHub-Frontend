## Features

- User registration and login
- Distributor registration and login
- Product and category management
- Order creation and management
- Quotation and pricing workflows
- Admin, customer, and distributor dashboard pages

## Tech Stack

### Frontend
- PHP
- HTML, CSS, JavaScript
- XAMPP (Apache)

### Backend
- C#
- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQL Server
- Swagger

## Run Frontend

1. Copy this project into your XAMPP `htdocs` directory.
2. Start Apache from XAMPP Control Panel.
3. Open:
   - `http://localhost/SOC44_Front_Ransara_GadgetHub_New/Homepage.php`

## Backend Repository

Backend API is maintained separately in C#:
